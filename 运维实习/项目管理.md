
项目类型：
```
本质上只有两类项目：  
✔ 内部项目  
✔ 外包项目（对外做的）  
而投标只是外包项目的一种来源方式
```

关于外包：
```
劳务外包（Labor Outsourcing）

乙方按照合同提供劳动力，由甲方指挥或按约定完成工作，通常不提供材料

人可以在 甲方现场 或 乙方自己场地 完工

既可以做体力活，也可以做技术服务

适用行业：建筑、制造、软件开发、运维服务、客服外包等


驻场（On-Site Service / Staff Augmentation）——劳务外包的子集

乙方人员必须在甲方现场办公，由甲方直接指挥调度

外包公司按“人天价”或“人月价”报价

常用于 IT 行业

外包员工

例如：
程序员驻场开发

运维工程师驻场值班

客服外包团队驻场话务室


两者的区别在于： 员工是否在甲方现场


甲方不直接招聘，而要“驻场外包”？？？

想承担正式员工的成本和风险

更灵活，可随时增减人手

不占编制（尤其是大企业、国企）

省成本：有时比正式员工更便宜；外包比正式工便宜 20%–40%，企业当然选择外包

法律合规性问题


```
