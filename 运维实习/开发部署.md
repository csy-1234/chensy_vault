nacos
一个「成熟团队」的 Nacos 架构长这样（重点）（命名空间隔离）

【开发环境】
  nacos-dev
    ├── dev-chen
    ├── dev-li

【测试环境】
  nacos-test
    └── test

【生产环境】
  nacos-prod
    └── prod
配置：
```
spring:
  cloud:
    nacos:
      discovery:
        server-addr: 127.0.0.1:8848
        namespace: dev-chen
      config:
        server-addr: 127.0.0.1:8848
        namespace: dev-chen
```
