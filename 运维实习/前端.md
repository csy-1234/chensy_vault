浏览器工作原理
```
网页、网站、Web 服务器和搜索引擎的区别
网页web page
	url访问==资源==唯一网页
网站web site
	网页集合
	网页共享唯一域名
	网页相互连接
	主页
web服务器 web server
	托管一个或多个网站的的计算机
搜索引擎 search engine
	特定类型的网站
	帮助寻找网站的网页
	Google、Bing、Yandex、DuckDuckGo
	浏览器是接收、显示网页的软件
	搜索引擎是寻找其他网页的网站
	火狐浏览器使用google作为搜索引擎
```
web如何工作
```
输入网址（URL）并按下回车 Enter/Return）到操作结果呈现给你（例如，网站出现在你的浏览器中）之间，发生了很多事情：
1.览器从存储资源的 Web 服务器中请求资源，通过http技术
2.求成功，Web 服务器会将包含所需资源的 HTTP 响应发送回 Web 浏览器；在某些情况下，请求的资源将触发更多 HTTP 请求，这将导致更多响应
```
